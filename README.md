## 📖 History & Origins

**Solid Bomber Pro**, originally discovered being sold as a "premium" paid tool in underground cybercriminal forums, the distributed binaries were obfuscated and secretly laced with malware designed to compromise the very users buying it. 

We successfully reverse-engineered the executable, and stripped out the hidden malicious payloads. This repo is the sanitized, malware-free version of that script, now open-source and free for everyone to learn from.

---

## ✨ Features

- 🚀 **Asynchronous Processing:** Built with `asyncio` and `aiohttp` for ultra-fast, concurrent API requests.
- 📱 **Multi-Service Integration:** Exploits 12 different SMS OTP services + 1 Call Bombing service.
- 🎨 **Beautiful CLI UI:** Clean, color-coded terminal interface with real-time statistics and batch tracking.
- 🛠️ **Auto-Dependency Installer:** Automatically checks and installs required Python packages on startup.
- ✍️ **Custom SMS Payloads:** Inject custom sender names and messages via the integrated SMS gateway.
- 🇵🇭 **PH Number Normalization:** Automatically formats and validates Philippine (+63) phone numbers.
